## Application Name
Github-Replica

## Description
This application is a replica of github.
It allows the user to search for github users their followers and repositories.

## Operating
First,The user searches for a github user hence  the person being searched on must be on github and have an account.
After searching,it will display the github user details such as the followers,following and the public repositories.
Someone can also click to view the person's profile in github.

## Technology Used
I used angular and its cli to develop the website.
I also used my custom css and bootstrap to style the website.
